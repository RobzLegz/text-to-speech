# TTS v1
Text to speech synthesising algorythm (latvian)

### Run program

```
git clone https://github.com/spotorg/TTS
```

```
cd v1
```

```
python -m venv /path/to/venv
```

```
pip install -r requirements.txt
```

```
python app.py
```
